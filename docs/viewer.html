JSON Viewer

This JSON Viewer is a Shiny-based solution originally developed for the 3rd CDISC Dataset-JSON Viewer Hackathon for which it won an honorable mention. It allows you to interactively explore and analyze CDISC Dataset-JSON data. Credit goes to the original developers of the application. The repo for the app can be found here.
